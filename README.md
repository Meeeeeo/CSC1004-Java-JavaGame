# Java Game - McDonalismApp


To run this game, you need to open the project in an IDE (I used IntelliJ IDE).

## Steps:

1. Clone this repository:

2. Open the project in your IDE.

3. Locate and Run:

   ```
   src/main/java/testPackage/McDonalismApp.java
   ```

##  Notes

* Make sure your IDE has correctly set up the JDK.
* If there are dependencies, ensure they are properly configured.
